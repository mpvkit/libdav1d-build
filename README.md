# libdav1d-build

build scripts for [dav1d](https://github.com/videolan/dav1d)

## Installation

### Swift Package Manager

```
https://github.com/mpvkit/libdav1d-build.git
```

## How to build

```bash
make build
# or build specified platforms 
make build platform=ios,macos
# or see help
make help
```
